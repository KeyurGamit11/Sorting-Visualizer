# Sorting-Visualizer


A visual representation of sorting algorithms built with Python and Pygame. 

## Features
- Bubble Sort
- Insertion Sort
- Merge Sort

## How to Run
1. Install the required libraries: `pygame`.
   ```bash
   pip install pygame
   
2.Run the Program:
   python main.py
   
3. Clone this repository:

   ```bash
   git clone https://github.com/your-username/SortingVisualizer.gi   
Controls
R: Reset the list
SPACE: Start Sorting
A: Ascending order
D: Descending order
B: Bubble Sort
I: Insertion Sort
M: Merge Sort
